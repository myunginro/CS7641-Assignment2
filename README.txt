RandomOptimization.py: runs RHC, SA, GA, and MIMIC for 3 optimization problems (OneMax, 4 Peaks, Max K Color)
NN_Optimization.py: optimizes NN using GD, RHC, SA, and GA for Breast Cancer Classifier

Run the file RandomOptimization.py and NN_Optimization.py, which will generate the figures used in the analysis under results/{filename}.png

datasets: taken from Scikit-learn:
    Breast Cancer: https://scikit-learn.org/stable/modules/generated/sklearn.datasets.load_breast_cancer.html#sklearn.datasets.load_breast_cancer

optimization problems and algorithm implementations used from MLROSE:
    algorithms: https://mlrose.readthedocs.io/en/latest/source/algorithms.html
    problems: https://mlrose.readthedocs.io/en/stable/source/fitness.html
